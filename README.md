### Project Description
The Smart Health Assistant is a Tkinter-based desktop application for basic personal health assessment.
It calculates BMI, evaluates hydration, sleep, and activity levels, and generates a health score.
The app provides instant feedback and simple recommendations to promote healthier lifestyle habits.

### The Problem statementT
The problem of lack of basic personal health awareness and helps users quickly check their BMI, hydration, and lifestyle habits.
It provides instant feedback to encourage healthier daily decisions and preventive care.
